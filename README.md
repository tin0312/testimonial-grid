## My process

- Build the entire grid layout for desktop screen.
- Use media query to specify otherwise css rules to handle mobile views.
- Edit and prepare readme file accordingly.
- Deploy the page on netlify.

### Built with

- Semantic HTML5 markup
- CSS custom properties
- Flexbox
- CSS Grid
- Media query for responsive design

### What I learned

- Use grid 2 dimensional layout system to structure the testimonials section in a more organized way, making it easier to read and visually appealing.
- Use media query to make the grid layout responsive on mobile devices.
- Use grid-row / grid-column and grid-template to size and place items on the grid and for responsive design.
